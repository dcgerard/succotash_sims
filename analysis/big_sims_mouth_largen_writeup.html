SUCCOTASH vs Vanilla ASH n = 10,000

David Gerard

2016-03-16

Abstract

I compare SUCCOTASH with ASH under different variance models.

Methods

I first normalized the counts by \(\log(COUNTS + 1)\). The number of hidden confounders was estimated using the methods of Buja and Eyuboglu (1992) implemented in the num.sv() function in the sva package in R.

I ran SUCCOTASH with PCA using either a homoscedastic or heteroscedastic model. I included the 75th percentile of log(counts + 1) as a covariate to adjust for library size.

I also ran vanilla ASH using OLS estimates, also with the 75th percentile of log(counts + 1) as a covariate, using either a homoscedastic or heteroscedastic model.

Simulation Study

I ran through 100 repetitions of generating data from GTEX lung data under the following parameter conditions:

  • \(n \in \{10, 20, 40\}\),
  • \(p = 10000\),
  • \(\pi_0 \in \{0.5, 0.9\}\),
  • \(\sigma_{log2} = 1\).

I extracted the most expressed \(p\) genes (excluding the top 5 expressed genes) from the GTEX lung data and \(n\) samples are chosen at random. Half of these samples are randomly given the “treatment”" label 1, the other half given the “control” label 0. Of the \(p\) genes, \(\pi_0p\) were chosen to be non-null. Signal was added by the Poisson-thinning approach in Mengyin’s code with a mean log2-fold change of 0 and a standard deviation log2-fold change of \(\sigma_{log2}\). That is

\[A_1,\ldots,A_{p/2} \sim N(0,\sigma^2_{log2})\\ B_i = 2^{A_i} \text{ for } i = 1,\ldots, p/2.\]

If \(A_i > 0\) then we replace \(Y_{[1:(n/2), i]}\) with \(Binom(Y_{[j,i]}, 1 / B_i)\) for \(j = 1,\ldots, n/2\). If \(A_i < 0\) then we replace \(Y_{[(n/2 + 1):n, i]}\) with \(Binom(Y_{[j,i]}, B_i)\) for \(j = n/2 + 1,\ldots, n\).

For each iteration, I calculated two things:

  1. The AUC using the lfdrs.
  2. The estimates of \(\pi_0\).

Results

Methods using homoscedastic assumption are better calibrated than methods using a heteroscedastic assumption. SUCCOTASH generally has higher AUC than vanilla ASH. Heteroscedastic models generally have higher AUC than homoscedastic models.

Heteroscedastic SUCCOTASH has the highest AUC but the worst calibration.
